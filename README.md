# TFT Streamlit Dashboard

This app shows:
- Top 10 replenish recommendations
- Top 10 promote recommendations
- Actual vs forecast plot by store and item

## Files
- `app.py` - Streamlit app
- `requirements.txt` - dependencies
- `artifacts/` - exported dashboard data from Colab

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
